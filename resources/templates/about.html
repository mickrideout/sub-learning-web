Sublearning is a flash card learning site for learning languages using movie/show subtitles. It has subtitles in 62 languages with 3106 combinations of these languages.


Once you select two languages, sublearning will display a line from a random subtitle. You should try and guess what the translation for that line is. Then click show answer to reveal the equivalent line in the second language.
